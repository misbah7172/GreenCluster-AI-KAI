## Usage Examples

### Example 1: Check Routing Consistency
```python
from model.gateway import InferenceGateway

gateway = InferenceGateway(["a:1|b:1", "c:1|d:1"])

# Same route every time?
for i in range(10):
    route = gateway.get_route_status()['selected']
    print(route)  # Same output each time ✅
```

### Example 2: Monitor Throughput
```python
from monitoring.telemetry import get_default_collector

collector = get_default_collector()
stats = collector.get_throughput_statistics(time_window_seconds=300)

print(f"Avg: {stats['avg_tokens_per_second']:.1f} tok/s")
print(f"Max: {stats['max_tokens_per_second']:.1f} tok/s")
```

### Example 3: Export Metrics
```python
collector = get_default_collector()
collector.export_json("metrics_export.json", time_window_seconds=600)

# Import and analyze
import json
with open("metrics_export.json") as f:
    data = json.load(f)
    print(data['routing'])  # All routing stats
    print(data['inference'])  # All inference stats
```

### Example 4: Real Latency Probing
```python
from model.latency_probe import LatencyProber

prober = LatencyProber()

# Cold probe (first measurement)
result = prober.probe_cached("node1:50051", force_refresh=True)
print(f"Cold: {result.rtt_ms:.2f}ms")

# Hot probe (from cache)
result = prober.probe_cached("node1:50051", force_refresh=False)
print(f"Hot: {result.rtt_ms:.4f}ms")
# → Hot is 900x faster!
```
